# Google Maps Scraper

A web scraper built with **Selenium** and **Python** to extract company information from Google Maps. The project includes a simple **Streamlit** frontend to input search queries, control result count, and display/export scraped data.

---

## Features

- Search companies/businesses on Google Maps by keyword
- Scrape business name, address, and phone number
- Headless Chrome browser automation using Selenium
- Interactive UI built with Streamlit
- Download results as CSV
